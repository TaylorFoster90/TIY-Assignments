### 1. `<header>` -- Header 
* *type:* block-level
* *content:* Flow content/palpable content
* *support:* HTML 5 
* *example:* 
```
 <header> Logo </header>
```
<header> Logo </header>

### 2. `<hr>` -- Horizontal Rule
* *type:* block-level
* *content:* Flow Content
* *support:* All HTML
* *example:*
``` <p> Some text </p>
 <hr>
<p> some other text </p> ```
<p> some Text </p>
<hr>
<p> some other text </p>
